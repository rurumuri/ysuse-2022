# 资源分享

这里分享一些笔者尝试过或接触过的、认为可能有帮助的网络资源链接。

:::warning 叠甲

笔者不负责任地(￣▽￣ = ￣︿￣)把本科阶段接触过的，自认为可能对你有帮助的资源链接分享在了这里，然而：

1. 这些资源未必适合你：打个比方，不是每个人都要精通编译原理，都要手搓操作系统、编译器和游戏引擎。如果你刚刚进入大学，时间充裕，大可以广泛寻找更多资源，探索自己感兴趣的方向；而如果你已经有了明确的选择，那么选择性探索，集中精力或许会有更好的收获。

2. 这些资源并不全面：它们大多是公开的可被搜索的网络资源，而其他渠道（比如微信公众号等自媒体）也有很多有价值的资源。许多资源信息也可能只是同学间小范围传播，有些经验也必须亲历才能获得。

3. 这些资源具有时效性，而时代每天都在变化：如今，AI绝对可以应付燕山大学的绝大多数课程项目了（比如那个记账本），而菜鸟教程和C++实验指导书可能还在教你如何一步一步配置环境直到输出一行“Hello World”。请善用AI和搜索引擎~

:::

## 计算机学习

### Linux

- [Linux 101](https://101.lug.ustc.edu.cn/)：个人认为是相比于《鸟哥的Linux私房菜》更现代化的Linux初学者教程
- [Kernel Exploring](https://richardweiyang-2.gitbook.io/kernel-exploring)：相对进阶的Linux内核探索指南，可以给你一些内核探索方向
